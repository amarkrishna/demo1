Magpie Manual

What is Magpie?

Magpie is an acronym for “Material-Agnostic Platform for Informatics and Exploration”, and is named after an intelligent bird.

The overall goal in developing Magpie is to create a single platform for performing tasks that involve the application of machine learning to the design of new materials. This software package enables users to create, test, and share machine learning models that can be used for a broad range of applications.

Reasons to use Magpie

  1. Ability to manipulate datasets, create models, and evaluate performance in the same program
  2. Access to extensive library of machine learning algorithms through Weka
  3. Interactive text-based scripting interface
  4. Easy sharing of models in a machine-independent format

Getting Magpie

Magpie is freely available for download. The distribution of Magpie includes:

  • All libraries necessary to run Magpie
  • This manual, and documentation for the Java interface
  • Example scripts for several common tasks

Magpie is written entirely in Java, which allows it to be run on most computers without modification. As long as the Java 7 Runtime Environment is installed on your computer, Magpie should run without issue.
